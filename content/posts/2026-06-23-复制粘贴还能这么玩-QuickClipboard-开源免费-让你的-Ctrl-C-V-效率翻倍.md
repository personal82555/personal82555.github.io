## QuickClipboard 到底是什么来头？

QuickClipboard 是由开发者 mosheng1 打造的一款跨平台剪贴板增强工具，基于 **Tauri 2 + Rust + React** 构建。目前支持 Windows 和 Android 平台，启动即常驻系统托盘，你复制的一切它都在后台默默记着。

和那些用 Electron 写的工具不一样，Tauri 2 + Rust 的组合让 QuickClipboard 的**内存占用极低** ，实测在后台运行时只有 40MB 左右，几乎感觉不到它的存在。

## 核心功能一：全类型剪贴板历史

这是它的看家本事。自动保存**文本、HTML 富文本、图片、文件** ，所有你复制过的东西它都帮你记住。支持智能去重和搜索筛选，按快捷键就能呼出历史面板，找到需要的复制内容，点击即可粘贴。

内置虚拟桌面分组，工作和娱乐可以分开，不会把公司文档和个人聊天记录混在一起。所有数据通过 SQLite 持久化保存，重启电脑也不会丢失。

## 核心功能二：截图 + OCR 一步到位

内置截图工具，一键截图后可以标注、添加水印，自动存入剪贴板历史。配合 **OCR 文字识别** ，图片上的文字都能一键提取出来复制。

这个功能太实用了——比如你在网上看到一张截图里有一段重要的文字信息，以前得一个字一个字打出来，现在直接截图→OCR→复制，几秒钟搞定。

## 核心功能三：局域网同步 / WebDAV 备份

这是 v0.4.0 版本的亮点功能！可以把剪贴板内容同步到多台设备上。办公室电脑复制的内容，回家笔记本上直接粘贴，无缝衔接。

通过 WebDAV 还能备份到 NAS 或云盘，剪贴板数据多重备份，再也不用担心丢失重要内容了。

## 核心功能四：个性化定制

支持亮暗主题切换，多套主题风格任你选——浅色、暗色、手绘风，甚至还能自定义背景和模糊效果。窗口位置、大小、快捷键全部可自定义。

用 Rust 原生性能驱动，启动飞快，后台运行完全无感。界面颜值在线，毛玻璃效果看起来很舒服，不像有些工具那样丑得让人不想打开。

## 和同类工具比怎么样？

市面上的剪贴板管理工具不少，我简单对比了一下：

**Ditto** — 老牌工具，功能稳定但只有 Windows 版，没有同步功能，界面比较老旧。

**ClipClip** — 功能全面但 Pro 版要 99 块，而且也是仅限 Windows。

**CopyQ** — 开源跨平台，功能强大但学习成本高，界面不太友好。

**QuickClipboard** — 开源免费，Windows+Android，自带同步+OCR，颜值在线，对新手友好。

QuickClipboard 在同步和 OCR 这两个维度上明显领先于同类工具，而且还免费开源，没有功能阉割，这点必须点赞。

## 优点与需要注意的地方

**✅ 优点：**

基于 Rust + Tauri 2 构建，启动快、内存低；界面颜值在线，毛玻璃效果看着舒服；局域网同步太实用了，办公室和家里无缝切换；截图 + OCR 一体化，省掉一个独立截图工具。

**⚠️ 需要注意：**

依赖 WebView2 运行时（Windows 11 自带，Win10 可能需要手动安装）；Android 版还在持续开发中；插件和脚本扩展能力不如 Ditto/CopyQ，适合轻度到中度用户。
