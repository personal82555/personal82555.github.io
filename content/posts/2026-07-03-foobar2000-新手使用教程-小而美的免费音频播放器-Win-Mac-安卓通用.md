## 为什么你需要一款专门的音频播放器？

"电脑上听歌到底用什么软件？"——这个问题我最近被问了好几次。

Windows自带的媒体播放器吧，格式支持太少，Flac放不了。装QQ音乐网易云？界面是好看，但动不动弹窗推广，内存占得飞起，还非得登录才能听本地歌。

更别提那些来路不明的"破解版播放器"了，下载下来捆绑一堆流氓软件，用得提心吊胆。

直到我翻出老玩家们口口相传的神器——**foobar2000** 。这家伙我大学就在用，十多年过去了，它还在更新，还是那么小巧、干净、能打。

## foobar2000 到底是什么？

简单说，foobar2000 是一款**极度轻量** 、**完全免费** 的音频播放器，由瑞典开发者 Peter Pawłowski 从2002年开始维护至今。

它支持市面上几乎所有音频格式：MP3、FLAC、APE、OGG、AAC、WAV、ALAC、Opus……你电脑里存的音乐文件，基本没有它播不了的。

而且它同时支持 **Windows、macOS、Android、iOS** 全平台，手机和电脑都用同一个 App，体验很统一。

## 第一步：下载和安装

下载很简单，直接去官网 **foobar2000.org** ，首页就是下载链接。

Windows用户选 64-bit 版本（现在电脑基本都是64位了），下载后双击安装，一路点"Next"就行。安装过程可以全部默认，不需要改什么设置。

![foobar2000 官网截图](api/attachments/w02aIMEiTkZO/image/6a469175e1bf4.png)

装完后打开，你会看到一个非常"朴素"的界面——白底黑字，列表样式，没有花里胡哨的皮肤。别被它劝退了，这正是它强大的地方：**什么资源都不占** ，打开秒启动。

## 第二步：把音乐加进播放列表

用 foobar2000 播歌有两种方式：

**方法一：直接拖拽。** 打开你的音乐文件夹，选中想听的歌，拖进 foobar2000 窗口，松手就播了。就这么简单。

**方法二：添加到媒体库。** 点击菜单栏 File → Preferences → Media Library，把存放音乐的文件夹加进去。之后在左侧栏的 Library 标签下，就可以按专辑、歌手、流派来浏览所有歌曲。

播歌的时候右键点击曲目，还能看到详细的音频信息——码率、采样率、声道数、编码器……对音质有要求的用户会很喜欢。

## 第三步：装插件，让它更好用

foobar2000 最大的魅力在于它的**插件系统** 。默认界面虽然朴素，但装几个插件之后，颜值和功能都能秒变专业级。

几个必装插件推荐：

  * **Lyric Show Panel 3** — 加载歌词插件，播放时自动显示滚动歌词。LRC格式和外置歌词文件都支持。
  * **Waveform Seekbar** — 在进度条位置显示音频波形，拖拽定位时可以看波形找高潮部分，很直观。
  * **Columns UI** — 替换默认界面，支持分栏、封面图显示、分组分类，让播放器看起来更现代。

安装方法：点击 File → Preferences → Components → Install，选择下载好的 .dll 或 .fb2k-component 文件，重启软件即可。

## 第四步：音质设置（小白也可以调）

foobar2000 默认输出用的是 DirectSound，如果你有稍微好一点的耳机或音箱，可以改成 WASAPI（独占模式），音质会有一点提升：

File → Preferences → Playback → Output → Device 里选择 "WASAPI (event)" 开头的设备。注意：选了 WASAPI 后，其他 App 就暂时出不了声了，听歌时可以独占声卡。

如果你是普通板载声卡+几十块的耳机，听不出太大区别，用默认设置也完全OK。

## 手机版：foobar2000 for Android

安卓用户有福了，foobar2000 的安卓版同样免费、无广告。去 Google Play 或官网下载 APK，安装后扫描手机里的音乐文件就能用了。

UI 比桌面版好看不少，支持专辑封面、歌词同步、睡眠定时，插耳机时自动播放。可以说是安卓上最干净的本土音乐播放器之一。
