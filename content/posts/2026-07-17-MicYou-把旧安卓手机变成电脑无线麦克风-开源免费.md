## MicYou 是干嘛的？

一句话：**让你的安卓手机充当电脑的无线麦克风** 。支持 Windows、macOS 和 Linux，连接方式包括 Wi-Fi、USB 和蓝牙。

开源项目，GitHub 上完全免费，手机端装 App，电脑端装服务端，配对就能用。

## 🎤 核心功能

**① 三种连接方式，总有一种适合你**

  * **Wi-Fi 模式** ：手机和电脑连同一个 Wi-Fi，无线连接最方便
  * **USB 模式（ADB/AOA）** ：用数据线连，延迟最低最稳定
  * **蓝牙模式** ：不用 Wi-Fi 也不用线，适合开会时用

**② 音频处理太良心了**

内置了 **噪声抑制** 、**自动增益控制（AGC）** 和 **去混响** 。什么意思呢？就是你手机放在桌上，旁边有点键盘声、空调声，它都能帮你过滤掉，录出来的人声干净很多。

**③ 可作为系统麦克风**

配合 VB-Cable 之类的虚拟音频线，MicYou 的信号可以作为系统的麦克风输入。这样你在任何软件——腾讯会议、OBS、微信、QQ 语音——里面都能用手机当麦克风。

## 📱 实际用下来咋样？

我拿了一台旧小米 6 试了一下，Wi-Fi 模式下延迟大概在 100-200ms 之间，日常语音通话完全够用。USB 模式延迟更低，几乎感觉不到。录音质量比电脑自带的麦克风好太多了——同样的距离，手机录出来清晰很多，背景噪声也少。

如果你是做播客、录课程、或者经常开会需要发言，这玩意儿真的能救急。而且旧手机放着也是吃灰，废物利用不香吗？

## 💻 安装步骤

**手机端：**

在安卓手机上安装 MicYou App，打开后选择连接模式。

**电脑端：**

从 GitHub Releases 下载对应系统的客户端：

  * Windows：下载 MicYou-Win 安装包
  * macOS：下载 .dmg 文件安装
  * Linux：下载 .deb 或 .rpm

macOS 用户需要额外安装 blackhole-2ch 和 switchaudio-osx 才能作为系统麦克风使用。  
`brew install blackhole-2ch --cask`  
`brew install switchaudio-osx --formulae`

装好后，手机和电脑配对，打开电脑端开始接收音频，搞定了。
