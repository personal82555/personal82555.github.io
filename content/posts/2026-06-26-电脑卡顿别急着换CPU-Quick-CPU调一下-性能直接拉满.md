## 🎯 Quick CPU 是干嘛的？

Quick CPU 是一款 CPU 性能调优与监控工具，支持 Intel 和 AMD 全系列处理器。它能实时监控 CPU 的温度、频率、电压等关键指标，还能让你微调核心停车、频率缩放和睿频策略。

说人话就是：**它能让你的CPU在需要性能的时候全力以赴，在不需要的时候省电降温。**

## 🚀 核心功能一看就懂

### 1\. 频率调整——让CPU跑得更快

Quick CPU 可以手动调整 CPU 的运行频率。比如你打游戏时，可以让它全速跑，保证帧率稳定；办公时适当降频，省电又凉快。

### 2\. 电压控制——找到性能和温度的平衡点

降低电压可以减少发热和功耗，但降太多会导致不稳定。Quick CPU 帮你找到那个"刚刚好"的点，让CPU在更低温度下跑出更高性能。

### 3\. 核心停车管理——让资源用在刀刃上

现在CPU都是多核心的。Quick CPU 可以管理哪些核心在工作、哪些在休息。需要高性能时唤醒所有核心，平时让部分核心休眠省电。

### 4\. 实时监控——一目了然

打开软件就能看到 CPU 温度、频率、使用率等实时数据。哪个核心在摸鱼、哪个核心在拼命，清清楚楚。
