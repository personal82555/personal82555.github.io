刚学编程或者平时需要写代码的朋友，肯定纠结过一个问题：用什么编辑器好？用记事本吧，连个行号都没有，代码高亮更别想了。用VS Code吧，又觉得杀鸡用牛刀，启动慢、吃内存、更新频繁。有没有一种编辑器——轻量、启动快、又能满足代码编写需求的？

答案就在今天要推荐的 **EditPlus** 。它是一款在Windows上存在了二十多年的老牌文本编辑器，适合写代码、做笔记、编辑配置文件，轻巧到只有几MB，打开速度比记事本还快。

## EditPlus是什么？

EditPlus 是由韩国 ES-Computing 公司开发的文本编辑器，支持 HTML、PHP、JavaScript、Python、C/C++ 等几十种编程语言的语法高亮。它的界面是经典的多标签页模式，可以同时编辑多个文件，不像记事本那样开一个文件就要开一个窗口。

**第三步：设置关联**  
第一次打开后，EditPlus 会问你要关联哪些文件类型。建议勾上 .txt、.html、.css、.js、.php、.py 这些你常用的格式，以后双击这些文件就会自动用 EditPlus 打开。 

## 新手必学操作

**语法高亮** — 打开任何代码文件，EditPlus 会自动识别语言并高亮显示关键字、字符串、注释等。再也不用在一堆黑白文字里找代码了。

**多标签编辑** — 按 `Ctrl+N` 新建文件，`Ctrl+Tab` 切换标签。可以同时打开几十个文件，互不干扰。

**查找替换进阶** — 按 `Ctrl+F` 打开查找，支持正则表达式和十六进制查找。替换时还能选择「所有打开的文件」，批量替换非常方便。

**网页预览** — 写HTML代码的时候，按 `Ctrl+B` 直接在浏览器中预览效果，写前端代码的利器。

**FTP功能** — 内置FTP客户端，可以远程连接服务器编辑文件，保存时自动上传，适合修改网站代码。

## 适合哪些人用？

**前端初学者** — 写HTML/CSS/JavaScript，EditPlus 的语法高亮和预览功能完全够用，不用启动 VS Code 那么大个家伙。

**运维人员** — 经常需要编辑配置文件（如 nginx.conf、.env 等），EditPlus 启动秒开，比 VS Code 方便太多。

**写作者** — 写 Markdown、LaTeX 等纯文本格式，EditPlus 干净无打扰，比 Word 省心。

**小技巧：** 在「首选项 → 字体」里可以设置等宽字体（推荐 JetBrains Mono 或 Consolas），写代码更舒服。在「首选项 → 文件」里可以设置默认编码为 UTF-8，避免中文乱码问题。

说实话，EditPlus 虽然不是一个"全能型"的编辑器（没有 VS Code 那种插件生态），但它胜在轻量、快速、稳定。如果你只是需要写点代码、编辑配置文件，完全没必要启动一个几百MB的编辑器。小而美，有时候就是最合适的。
