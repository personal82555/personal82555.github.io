## EditPlus 是什么？

EditPlus 是一款 Windows 平台的文本编辑器，别看它身材小（安装包才几 MB），功能可一点都不含糊。支持代码高亮、多标签编辑、FTP 上传、HTML 预览、正则搜索替换等等。跟那些动辄几百 MB 的「重型 IDE」比起来，EditPlus 就像一把瑞士军刀——随身携带、随叫随到。

## 一步步教你用 EditPlus

### 📥 下载安装

拿到安装包后双击运行，一路下一步就行。装完后打开，默认是英文界面，想用中文的话点菜单 `View → Language → Chinese Simplified`，界面就变成中文了。

我这里用的是烈火汉化版，深度汉化程度很高，连那些冷门的对话框都有中文，对新用户特别友好。

### 📝 新建和编辑文件

按 `Ctrl+N` 新建文件，会弹出文件类型选择——HTML、C/C++、Java、PHP、Perl、Python、JavaScript 等等。选好后编辑器会自动启用对应语言的语法高亮，关键字、字符串、注释都会用不同颜色标出来，代码结构一目了然。

EditPlus 用的是多标签界面，每个文件一个标签，可以同时打开几十个文件不卡顿。标签栏还支持左对齐排列，找文件很方便。

### 🔍 搜索和替换

按 `Ctrl+F` 打开搜索框，支持普通搜索和正则表达式搜索。如果你要在一大段代码里找某个模式，正则搜索绝对是救命的功能。比如搜 `\d{3}-\d{4}` 可以找到所有电话号码格式的文本。

按 `Ctrl+H` 是替换，支持批量替换。有个小细节：替换对话框里的「更多」按钮在 DPI 缩放后也正常工作了（v6.1 修复了这个问题）。

### 🌐 HTML 预览

写网页的时候，按 `Ctrl+B` 就能在浏览器里预览当前编辑的 HTML 文件，不用手动保存再打开浏览器刷新。写前端代码做快速原型测试非常实用。

### 📂 FTP 功能

这是 EditPlus 的隐藏大招——内置 FTP 客户端！菜单 `文件 → 打开 → FTP`，填上服务器地址、用户名密码，就能直接编辑服务器上的文件。改完按保存，自动上传到服务器，不用本地下载再上传，省事不少。

新版 v6.1 还支持了 MLSD FTP 命令，兼容性更好。FTP 上传对话框现在会自动预填上次用的账号和目录，非常人性化。

### 🛠️ 其他实用功能

  * **拼写检查** ：按 `Ctrl+Shift+S` 可以对所有打开的文件做拼写检查，写英文文档时有用
  * **代码折叠** ：可以折叠/展开代码块，看长文件时清爽很多
  * **自动完成** ：输入关键字时自动补全，提高编码速度
  * **语法着色模板** ：烈火版新增了 Python 等语言的着色模板，开箱即用

## 实际使用感受

说实话，EditPlus 不像 VS Code 那样有海量插件生态，但它的核心优势就是「快」和「轻」。开机启动 0.5 秒，打开几百 KB 的文件不用等，改完保存关闭，整个过程行云流水。如果你只是要改配置文件、写简单的脚本、编辑 HTML 页面，EditPlus 绝对比那些重型 IDE 舒服得多。

而且 v6.1 修复了不少 Bug，比如之前 SFTP 连接某些服务器会崩溃的问题、INI 文件在多个实例下损坏的问题，现在都修好了，用起来更稳了。
