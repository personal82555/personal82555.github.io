##  什么是 CrossPaste？ 

    简单说：**电脑复制，手机直接粘贴；手机复制，电脑直接粘贴** 。不需要发微信、不需要数据线、不需要任何中转操作。 

    它是一款开源免费的跨设备剪贴板同步工具，支持 Windows、macOS、Linux、Android 和 iOS 全平台。只要你的设备在同一个局域网（连同一个 WiFi），就能实现毫秒级的剪贴板同步——复制的瞬间，另一台设备就能粘贴。 

    关键是它完全免费，没有广告，没有会员制，数据全部本地存储，端到端加密保护隐私。在 GitHub 上已经收获了 2200+ Star，被 GitHub 官方评为「Project of the Week」。

##  5大核心功能 

**1\. 实时同步，毫秒级触达**

    电脑上 Ctrl+C 复制一段文字，手机上直接就能 Ctrl+V 粘贴。没有延迟，没有等待，复制的瞬间就同步过去了。甚至图片和文件也能同步——截个图直接在手机上粘贴发送，省去了好几步操作。 

**2\. 全平台覆盖，统一体验**

    Windows、macOS、Linux、Android、iOS 五大平台全覆盖。不管你用的是什么设备组合——Windows 电脑+安卓手机、Mac+iPhone、甚至 Linux 服务器——都能无缝衔接。而且每个平台的界面和操作逻辑都是一致的，不用重新学习。 

**3\. 丰富的内容类型支持**

    不只是文字——颜色代码、网页链接、HTML 富文本、RTF 格式、图片、文件，统统都能同步。每种类型还有专属的颜色标识，一眼就能分辨。写代码的时候复制一段高亮代码，粘贴到另一台设备上格式也不会乱。 

**4\. 端到端加密，隐私无忧**

    所有同步内容都经过端到端加密，数据只在你的设备之间传输，不会经过任何服务器。而且它只在局域网内工作，完全不需要联网——你的剪贴板内容永远不会离开你的网络。对于担心隐私的用户来说，这一点非常重要。 

**5\. 强大的剪贴板管理器**

    即使只在一台设备上使用，CrossPaste 也是一个功能完善的剪贴板管理器。快捷键快速调出历史记录、拖拽粘贴、无限历史、智能搜索——再也不用担心复制了新内容就把旧的覆盖掉了。内置 OCR 功能还能识别图片中的文字，截图后直接提取文本。

##  下载安装 

    安装非常简单，三步搞定： 

    **第一步** ：在电脑上下载安装 CrossPaste（Windows/macOS/Linux 都有对应版本）。Windows 用户也可以直接在 Microsoft Store 搜索「CrossPaste」安装。 

    **第二步** ：在手机上下载安装 CrossPaste（Google Play 或 App Store 搜索「CrossPaste」）。 

    **第三步** ：确保电脑和手机连在同一个 WiFi 下，打开两边的 CrossPaste，自动发现设备并配对。搞定！ 

    首次使用建议检查一下手机端的排序按钮，如果历史内容太多可能会把新粘贴的内容挤下去。另外 Chrome 浏览器用户还可以安装 Chrome 扩展，实现浏览器和手机之间的剪贴板同步。
