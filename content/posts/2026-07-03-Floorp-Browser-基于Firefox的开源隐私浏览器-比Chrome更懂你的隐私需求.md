## 受够了浏览器偷偷收集你的数据？

说实话，现在主流的浏览器或多或少都在"偷窥"你——Chrome 追踪你的浏览记录用于广告投放，Edge 也在学，连 Firefox 近年来也加了不少遥测。虽然对大部分人来说无所谓，但如果你比较在意隐私，那这款 **Floorp Browser** 可能正是你需要的。

## Floorp Browser 到底是什么来头？

Floorp（发音同"florp"）是一款基于 Firefox 开发的开源浏览器。它不是自己从头写的，而是站在 Firefox 的肩膀上，把 Firefox 里那些让普通用户头疼的东西简化了，同时保留了 Firefox 最核心的优势——强大的隐私保护能力。

![](https://img.88531.cn/i/2026/07/03/6a46940a18449.png)

## 为什么选 Floorp 而不是直接用 Firefox？

**🔒 隐私保护拉满**

Floorp 默认就开启了 Firefox 里最严格的隐私保护设置——阻止第三方跟踪器、阻止指纹识别、阻止加密矿工。开箱即用，不用自己折腾 about:config。用 Chrome 浏览网页时，你每访问一个网站，背后可能有几十个跟踪器在记录你的行为，而 Floorp 全部帮你挡掉了。

**⚡ 性能优化更好**

Floorp 团队对 Firefox 做了不少性能优化，启动速度更快，内存占用更低。特别是如果你电脑配置不太新，用 Floorp 会比 Chrome 流畅不少。

**🎨 高度可定制**

界面布局、标签页样式、工具栏——几乎一切都能自定义。你可以把 Floorp 改成 Chrome 风格、Edge 风格，或者完全自创一套。喜欢折腾的朋友有得玩了。

**📑 标签分组管理**

开几十个标签页不慌，Floorp 支持标签分组，类似 Edge 的标签页集合，把相关网页归到一个组里，需要的时候展开，不需要的时候收起来。

**🔌 兼容 Firefox 扩展**

Firefox 有的扩展，Floorp 都能装。uBlock Origin、Bitwarden、暴力猴……常用的扩展一个不少。

## 谁适合用 Floorp？

  * 比较在意隐私、不想被跟踪的用户
  * 电脑配置不高，想要轻量浏览器的用户
  * 喜欢自定义界面的"折腾党"
  * 对 Firefox 有好感但觉得原版用起来不够顺手的用户
